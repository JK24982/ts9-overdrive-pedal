# TS9 Overdrive Guitar Pedal

Analog guitar overdrive pedal inspired by the classic TS9 circuit.

## Overview

This project is a DIY guitar pedal based on a classic analog overdrive circuit using the JRC4558 operational amplifier.  
The pedal produces distortion using diode clipping and includes tone shaping typical for TS-style overdrive pedals.

## Features

- analog audio circuit
- JRC4558 operational amplifier
- diode clipping stage
- tone control
- true bypass switching

## Assembly

The circuit was assembled manually and mounted inside a stompbox enclosure with standard guitar pedal connectors.
